# 내 위치 기반 공공 와이파이 정보를 제공하는 웹서비스 개발

## 과제 상황
저는 자신의 위치를 가져오는 것과 공공 와이파이 정보를 가져와서 db에 저장하고 화면에 구현하는 기능까지만 하였습니다.

openAPI를 가져오는 것에서 많은 시행착오 끝에 가져오는 것을 성공했으나 제출기간이 얼마 남지않아서 나머지 부분을 구현하지 못했습니다.
하나를 구현하려면 구글링도 많이 해보고 유튜브에서도 참고를 많이 해서 성공하였습니다. 일부 기능만 구현하였지만 여러가지를 알게 되었습니다.
다음 과제에서는 더욱 분발된 모습을 보여드리겠습니다.
